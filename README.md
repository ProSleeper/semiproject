### - 개발환경: Java 1.8.0_333, eclipse ver. Photon, oracle db, querybox, apach-tomcat
### - 참여인원: 김영운, 이동우, 정영진, 안수언, 윤다영

# 담당 부분
### 김영운: Cart/payment page 기능 구현, DB관리, Source통합(Git)
